Projeto simples - Análise de Dados

Projeto desenvolvido na formação de Analista de Dados da EBAC.

Objetivo

Aplicar todas as etapas de um projeto de análise de dados, desde a seleção da base até a construção de dashboards e geração de insights para apoio à tomada de decisão.

Etapas desenvolvidas

Seleção dos Dados

* Escolha de uma base pública
* Integração de múltiplas tabelas
* Relacionamento entre datasets

Análise Exploratória

* Distribuição dos dados
* Correlação entre variáveis
* Identificação de outliers
* Estatísticas descritivas
* Análise de dispersão

Tratamento dos Dados

* Tratamento de valores nulos
* Correção de inconsistências
* Padronização de formatos
* Criação de colunas derivadas

Visualização

* Dashboard interativo
* Gráficos de barras
* Gráficos de linhas
* Gráficos de dispersão
* Tabelas dinâmicas

Ferramentas utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab
* Looker Studio
* GitHub

Arquivos

* `analise.ipynb`
* Bases de dados originais
* Bases tratadas
* Dashboard
* Relatório
